# 星川智盾 v1.11.0 发布说明

发布日期：2026-05-04

## 版本亮点

本版本聚焦于三件事：
1. 稳定性修复（类型系统与关键流程健壮性）
2. 大文件分析可观测性与性能基线建设
3. 自动化交付能力（E2E + CI + Release）完善

## 主要更新

### 1) 稳定性与兼容性修复
- 修复两个关键 TypeScript 类型错误，恢复主链路类型检查稳定性。
- 快捷键执行链路统一兼容同步/异步 action，避免 Promise 处理不一致。
- 搜索增强与交互式钻取逻辑兼容当前规则匹配结构，减少字段不一致导致的问题。

### 2) AI 分析可靠性增强
- 分析器新增截断启发式检测，不再依赖脆弱的固定标记判断。
- 增加自动续写机制，提升长文本分析结果完整性。
- 续写拼接采用重叠去重策略，减少重复内容并提升可读性。

### 3) 性能与大文件能力
- 新增性能追踪接入：记录分析耗时、Token 指标与处理统计。
- 新增日志性能基准脚本：
  - 命令：`npm run benchmark:logs`
  - 用途：评估大规模日志采样与处理吞吐，辅助容量规划。

### 4) 可观测性与诊断
- 新增本地诊断事件缓冲能力。
- 新增诊断快照导出功能（自动脱敏），便于定位问题和远程协作排障。

### 5) 测试与交付
- 新增 Playwright E2E 冒烟测试，覆盖核心界面与诊断导出流程。
- 新增 CI 校验流程：TypeScript 检查 + 前端构建 + E2E 冒烟测试。
- Release 按 `v*` 标签触发多平台构建并上传产物。

## 升级建议
- 建议从 v1.10.0 直接升级到 v1.11.0。
- 若在企业网络中使用 API 服务，建议配置代理并完成连通性检查。

## 已知说明
- 本版本不引入破坏性配置变更。
- 诊断导出为本地 JSON 文件，后续可按需扩展远程遥测能力。
