### 安装 & 运行

```bash
# 克隆项目
git clone https://github.com/chu0119/zhidun.git
cd zhidun

# 安装依赖
npm install

# 启动开发模式（热重载）
npm run dev
```

### 构建打包

```bash
# 构建当前平台安装包
npm run build

# 构建产物位于 release/ 目录
# Windows → .exe (NSIS 安装包)
# macOS   → .dmg 磁盘映像
# Linux   → .AppImage 便携应用
```

### 跨平台构建

项目已配置 GitHub Actions 自动构建，推送 `v*` 标签后自动在 Windows / macOS / Linux 三平台构建并发布：

```bash
# 打标签触发自动构建
git tag v1.6.0
git push origin v1.6.0
```

## `◈` 本地规则引擎

基于 **ModSecurity CRS v4** 和 **OWASP Top 10 2021** 标准，**65+ 条高置信度规则**覆盖 **19 个攻击类别**：

<table>
<tr>
<td width="50%">

| 类别 | 风险 | OWASP |
|:---|:---:|:---:|
| SQL 注入 | `🔴` 危急 | A03 |
| XSS 攻击 | `🔴` 危急 | A03 |
| 命令注入 | `🔴` 危急 | A03 |
| WebShell | `🔴` 危急 | A08 |
| 反序列化攻击 | `🔴` 危急 | A08 |
| 模板注入 (SSTI) | `🔴` 危急 | A03 |
| Log4j 注入 | `🔴` 危急 | A09 |
| Spring 漏洞 | `🔴` 危急 | A06 |
| SSRF 攻击 | `🔴` 危急 | A10 |
| 目录遍历 | `🔴` 危急 | A01 |

</td>
<td width="50%">

| 类别 | 风险 | OWASP |
|:---|:---:|:---:|
| 文件包含 | `🔴` 危急 | A03 |
| HTTP 请求走私 | `🔴` 危急 | A04 |
| HTTP 头注入 | `🔴` 危急 | A03 |
| JWT 攻击 | `🔴` 危急 | A02 |
| 敏感文件访问 | `🟠` 高危 | A01 |
| 攻击工具 | `🟠` 高危 | A05 |
| 信息泄露 | `🟡` 中危 | A05 |
| 暴力破解 | `🟡` 中危 | A07 |
| 爬虫 Bot | `🔵` 低危 | — |

</td>
</tr>
</table>

**设计原则**：

- 所有模式使用 `\b` 词边界，防止子串误匹配
- SQL 注入要求关键字组合（如 `union...select`），而非单个关键字
- SSTI 仅检测明确攻击特征（`{{7*7}}`、`{{__globals__}}`）
- 基于实际 Web 日志场景优化，最大限度降低误报率
