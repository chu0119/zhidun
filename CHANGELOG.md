# Changelog

## 1.12.5 - 2026-05-05

- 增强更新体验：补全更新检查、下载、安装流程，并支持自动更新提示。
- 修复世界地图与图表稳定性问题，改为按需加载 ECharts 相关模块以降低首屏体积。
- 统一时间线与攻击聚合语义，减少不同面板之间的数据偏差。
- 新增诊断导出与上传开关，默认保持关闭，需显式授权。
- 完善报表导出与 Markdown 解析，提升 DOCX / PDF 导出一致性。

## 1.12.4 - 2026-05-04

- 增强地理面板稳定性：地图注册失败后支持重试，避免地图区域空白。
- 增加地图注册失败诊断事件，便于定位 Geo 渲染问题。
- 完成 1.12.4 发布版本封版与构建同步。

## 1.12.3 - 2026-05-04

- 修复世界地图初始化时序问题，避免渲染阶段触发注册失败。
- 图表面板增加地图重试机制，降低首次加载失败率。
- 完成 1.12.3 版本发布打包。

## 1.12.2 - 2026-05-04

- 修复 `global` 兼容性问题，提升不同运行环境下的稳定性。
- 更新性能跟踪模块，降低因全局对象差异导致的运行错误。
- 完成 1.12.2 热修复发布。

## 1.12.1 - 2026-05-04

- 修复启动黑屏问题，提升应用冷启动稳定性。
- 补强 CI 与 E2E 测试配置，减少发布后回归风险。
- 完成 1.12.1 热修复发布。

## 1.12.0 - 2026-05-04

- 新增规则策略预设，优化不同场景下的规则引擎配置体验。
- 增加链式分析能力，增强从日志到威胁结论的关联分析。
- 同步更新设置面板、Worker 与主进程通信相关实现，发布 1.12.0 主版本。
